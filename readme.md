# SUS
wanted to sus-up your terminal?
if so - its for you

just: 
- copy "susdo" file to either /bin , /usr/bin or any other executable place
- check if susdo is executable (chmod 777 susdo)
- run susdo in your terminal

features:
- red terminal forever
- sus image at startup
- new passworded admin user (with sus'ed name/logo/image)

what do you wait for?

get sus'ed !
